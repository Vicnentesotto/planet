About This Project

The Solar System Explorer is an educational website designed to make learning about our cosmic neighborhood engaging and interactive. Our goal was to create a visually stunning resource that presents accurate scientific information in an accessible way.

This project was developed as part of our Empowerment Technology course, where we applied web development skills to create meaningful educational content. Each planet page includes detailed information, high-quality NASA images, and interactive elements to enhance the learning experience.

We hope this website inspires curiosity about astronomy and our place in the universe. The solar system is vast and full of wonders, and we've tried to capture some of that majesty in this project.
